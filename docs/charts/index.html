KBeacon Helm Repository

This directory is the public Helm repository for the KBeacon chart.

Helm install

helm repo add kbeacon https://memoliyasti.github.io/kbeacon/charts
helm repo update
helm upgrade --install kbeacon kbeacon/kbeacon \
  --namespace kbeacon-system \
  --create-namespace \
  --set cluster.name=prod-eu-1
